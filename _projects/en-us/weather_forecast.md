This study focuses on understanding weather forecasting and climate modeling using machine learning and deep learning, and practically using one of the recently developed foundation model called ClimaX. In the four articles I published, I have explored the difference between weather forecast and climate modeling, and the motivation for why machine learning approach is needed in this field. Moreover, I have shown how to use ClimaX for a real forecast as an example Python code and notebook, and laid out a roadmap to use this foundation model for weather forecast/climate modeling in Japan.
